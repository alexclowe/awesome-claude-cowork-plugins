# Teacher Plugin for Claude

Build lesson plans, create rubrics, and draft parent communications.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/teacher) — AI tools, guides, and weekly digests designed specifically for teachers.

## Commands

| Command | Description |
|---------|-------------|
| `/lesson-plan` | Generate standards-aligned lesson plans with differentiation, assessments, and timing |
| `/rubric` | Create detailed grading rubrics (analytic, holistic, or single-point) aligned to learning objectives |
| `/parent-email` | Draft professional parent communications for progress updates, behavior concerns, and conferences |
| `/worksheet` | Generate differentiated practice materials with answer keys for any subject area |

## Skills

This plugin includes skills that activate automatically when you're working on education-related tasks:

- **Curriculum Design** — Backward design, standards alignment, Bloom's taxonomy, differentiated instruction, and UDL principles
- **Classroom Communication** — Parent partnership communication, student feedback, IEP/504 documentation support, and culturally responsive communication

## Usage examples

```
/lesson-plan
Create a 5th grade science lesson on the water cycle. Standards: NGSS 5-ESS2-1.
Include inquiry-based activities, differentiation for 3 levels, and a formative
assessment exit ticket. 50-minute class period.
```

```
/rubric
Create an analytic rubric for a 9th grade persuasive essay. Criteria: thesis
statement, evidence and reasoning, counterargument, organization, conventions.
4 performance levels (Exemplary, Proficient, Developing, Beginning). Align to
Common Core W.9-10.1.
```

```
/parent-email
Draft a positive behavior recognition email for a 3rd grader who has shown
significant improvement in classroom participation and collaboration over the
past month. Include specific examples the parent can celebrate at home.
```

```
/worksheet
Generate a differentiated math worksheet on multiplying fractions for 6th grade.
Include 3 levels: approaching (visual models), on-level (standard algorithm),
and advanced (word problems with multi-step reasoning). 10 problems per level
with answer key.
```

## Disclaimer

All educational content generated by this plugin is for planning and drafting purposes only. It does not constitute official curriculum or instructional policy. The teacher is responsible for reviewing all content and adapting it to their school and district policies, state standards, and individual student needs before classroom use.
