# Teacher Plugin for Claude

Build lesson plans, create rubrics, draft parent communications, and run AI-classroom guardrails (FERPA/COPPA, AI ethics literacy, exam-integrity review).

Built by [The AI Career Lab](https://theaicareerlab.com/professions/teacher) — AI tools, guides, and weekly digests designed specifically for teachers.

## Commands

| Command | Description |
|---------|-------------|
| `/lesson-plan` | Generate standards-aligned lesson plans with differentiation, assessments, and timing |
| `/rubric` | Create detailed grading rubrics (analytic, holistic, or single-point) aligned to learning objectives |
| `/parent-email` | Draft professional parent communications for progress updates, behavior concerns, and conferences |
| `/worksheet` | Generate differentiated practice materials with answer keys for any subject area |
| `/student-data-audit` | Ensure AI-assisted workflows comply with FERPA / COPPA / GDPR; flag PII risks and consent requirements |
| `/ai-ethics-brief` | Generate a classroom discussion guide on AI bias, hallucination, and academic integrity, tailored to grade level |
| `/exam-integrity-check` | Flag suspicious similarities between submissions without false-positives and produce evidence summary for honor-code review |

## Skills

This plugin includes skills that activate automatically when you're working on education-related tasks:

- **Curriculum Design** — Backward design, standards alignment, Bloom's taxonomy, differentiated instruction, and UDL principles
- **Classroom Communication** — Parent partnership communication, student feedback, IEP/504 documentation support, and culturally responsive communication
- **Classroom AI Ethics** — FERPA, COPPA, age-appropriate AI literacy framing, bias-discussion frameworks, and academic-integrity guardrails

## Usage examples

```
/lesson-plan
Create a 5th grade science lesson on the water cycle. Standards: NGSS 5-ESS2-1.
Include inquiry-based activities, differentiation for 3 levels, and a formative
assessment exit ticket. 50-minute class period.
```

```
/rubric
Create an analytic rubric for a 9th grade persuasive essay. Criteria: thesis
statement, evidence and reasoning, counterargument, organization, conventions.
4 performance levels (Exemplary, Proficient, Developing, Beginning). Align to
Common Core W.9-10.1.
```

```
/student-data-audit
I want to use Claude to help me draft IEP progress notes from my class roster.
The roster has student names, grade levels, IEP goals, and behavioral data
from the past quarter. Tool: my personal Claude account (not district-approved).
Audit this workflow against FERPA and our state law.
```

```
/ai-ethics-brief
Build a 2-class-period AI ethics unit for my 8th grade ELA class. Tie to ISTE
Standards 3 and 7. Focus on hallucination, bias, and academic integrity.
Include a hands-on AI fact-check activity and an exit ticket. Local context:
our district just adopted an AI policy that allows AI for brainstorming with
disclosure but prohibits AI-generated final submissions.
```

```
/exam-integrity-check
I have two essays from my AP US History class with very similar arguments
about the New Deal. The students sit next to each other. The assignment
asked for an original thesis with primary-source evidence. Both submissions
cite a quote from FDR that I can't find in any of the assigned sources.
Compare the submissions and flag what's worth a follow-up conversation.
[paste both essays]
```

```
/parent-email
Draft a positive behavior recognition email for a 3rd grader who has shown
significant improvement in classroom participation and collaboration over the
past month. Include specific examples the parent can celebrate at home.
```

## Disclaimer

All educational content generated by this plugin is for planning and drafting purposes only. It does not constitute official curriculum, instructional policy, or legal advice on student-data privacy. The teacher is responsible for reviewing all content and adapting it to their school and district policies, state standards, and individual student needs before classroom use. **Never share student PII with any AI tool without explicit district authorization and a Data Privacy Agreement in place.**
