# Veterinarian Plugin for Claude

Draft SOAP notes, write discharge summaries, and generate client communications.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/veterinarian) — AI tools, guides, and weekly digests designed specifically for veterinarians.

## Commands

| Command | Description |
|---------|-------------|
| `/soap-note` | Draft complete veterinary SOAP notes from examination findings with assessment and treatment plan |
| `/discharge-summary` | Generate client-friendly discharge instructions with medication schedules and follow-up care |
| `/referral-letter` | Create specialist referral letters with clinical history and specific consultation requests |
| `/client-email` | Draft treatment estimates, follow-up reminders, lab result explanations, and sensitive discussion guides |

## Skills

This plugin includes skills that activate automatically when you're working on veterinary-related tasks:

- **Veterinary Medicine** — Clinical documentation, diagnostics, pharmacology, treatment protocols, and species-specific medical knowledge
- **Pet Owner Communication** — Empathetic, plain-language communication for treatment discussions, costs, prognosis, and end-of-life care

## Usage examples

```
/soap-note
SOAP note for 7yo MN Golden Retriever. CC: limping on right forelimb x3 days.
PE: pain on palpation of right elbow, decreased ROM, grade 2/5 lameness. Rads:
osteophytes on anconeal process. Assessment: elbow dysplasia with secondary OA.
```

```
/discharge-summary
Discharge summary for feline dental extraction. Removed teeth #304, #307, #404.
GA with propofol induction, isoflurane maintenance. Meloxicam 0.1mg/kg SQ given.
Owner needs home care instructions for soft food diet and pain management.
```

```
/referral-letter
Referral letter to veterinary dermatologist. 4yo FS French Bulldog with chronic
bilateral otitis externa and ventral interdigital dermatitis. Failed empirical
therapy with Apoquel and topical antifungals. Requesting allergy workup and
advanced diagnostics.
```

```
/client-email
Treatment estimate email for client. Dog needs TPLO surgery for CCL rupture.
Include surgical estimate ($3,500-4,500), pre-op workup, recovery timeline,
and payment options.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute veterinary medical advice. The veterinarian is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
