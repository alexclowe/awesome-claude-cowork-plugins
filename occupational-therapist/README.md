# Occupational Therapist Plugin for Claude

Draft SOAP notes, treatment plans, progress reports, and home exercise programs for occupational therapy.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/occupational-therapist) — AI tools, guides, and weekly digests designed specifically for occupational therapists.

## Commands

| Command | Description |
|---------|-------------|
| `/soap-note` | Draft complete OT SOAP notes from session observations with functional assessments and intervention documentation |
| `/treatment-plan` | Generate individualized treatment plans with measurable goals, intervention strategies, and timelines |
| `/progress-report` | Create progress reports summarizing functional gains, goal status, and recommendations for continued care |
| `/home-program` | Produce illustrated home exercise and activity programs with clear instructions for patients and caregivers |

## Skills

This plugin includes skills that activate automatically when you're working on occupational therapy tasks:

- **Occupational Therapy** — ADL assessment, fine motor evaluation, sensory processing, cognitive rehabilitation, and hand therapy documentation
- **Patient Education** — Home program instructions, caregiver training materials, and adaptive equipment recommendations

## Usage examples

```
/soap-note
SOAP note for 68yo female s/p R CVA 3 weeks ago. Seen for UE ADL
retraining. Patient able to don upper body clothing with mod A using
adaptive techniques. L UE AROM shoulder flexion 0-95 degrees, grip
strength 12 lbs (R dominant, now using L). Demonstrated improved
sequencing for brushing teeth with setup assistance only.
```

```
/treatment-plan
Treatment plan for 5yo male with sensory processing disorder and fine
motor delays. Difficulty with handwriting, scissor use, and self-care
tasks (buttons, zippers). Sensory seeking behaviors in classroom.
Currently receiving OT 2x/week, 45-minute sessions. Parents want
school readiness goals for kindergarten.
```

```
/progress-report
Progress report for 45yo male, construction worker, 8 weeks post R
carpal tunnel release. Initial grip strength 18 lbs, now 42 lbs
(goal: 60 lbs for return to work). Thenar atrophy improving. Can now
tolerate sustained grip for 10 minutes (up from 2). Light duty work
trial started. Still reports numbness in index and middle fingers with
prolonged tool use.
```

```
/home-program
Home program for 78yo female with moderate dementia living with daughter.
Focus on maintaining current ADL independence: grooming, simple meal
prep, and medication management with cueing. Include caregiver strategies
for task simplification, environmental modifications, and a visual daily
schedule. Keep instructions simple with large print.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute medical or therapeutic advice. The occupational therapist is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
