# Construction Project Manager Plugin for Claude

Create project estimates, change orders, progress reports, and RFI responses for construction project management.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/construction-pm) — AI tools, guides, and weekly digests designed specifically for construction project managers.

## Commands

| Command | Description |
|---------|-------------|
| `/project-estimate` | Generate detailed project cost estimates with line items, labor, materials, contingency, and summary breakdowns |
| `/change-order` | Draft change order documentation with scope changes, cost impacts, schedule adjustments, and approval routing |
| `/progress-report` | Create weekly or monthly progress reports with milestone tracking, budget status, and risk assessments |
| `/rfi-response` | Compose professional RFI responses with technical clarifications, drawing references, and resolution documentation |

## Skills

This plugin includes skills that activate automatically when you're working on construction-related tasks:

- **Construction Management** — Estimating, scheduling, building codes, safety protocols, permitting, and project controls
- **Stakeholder Communication** — Owner reports, subcontractor coordination, RFI responses, and meeting documentation

## Usage examples

```
/project-estimate
Preliminary estimate for a 2,400 sq ft single-family residential addition.
Wood frame construction, new foundation, 2 bedrooms, 1 bathroom, open
living/kitchen area. Include site prep, foundation, framing, MEP rough-in,
insulation, drywall, finishes, and landscaping restoration. Region: Northeast US.
```

```
/change-order
Change order for the Riverside Office Building project. Owner requested
upgrading all restroom fixtures from standard to ADA-compliant touchless
units. Affects floors 2-5 (16 restrooms total). Need revised plumbing
rough-in, new fixture specs, and additional inspection. Original bathroom
budget was $124,000.
```

```
/progress-report
Monthly progress report for Oakmont Elementary School renovation. Month 3
of 8. Demolition complete, structural reinforcement 90% done, MEP rough-in
started on schedule. Budget at 34% spent of $2.8M total. One delay: HVAC
equipment delivery pushed 2 weeks due to supply chain. No safety incidents
this month.
```

```
/rfi-response
RFI response for the Harbor View Condos project. Subcontractor asking about
the waterproofing detail at the transition between the podium deck and
curtain wall system. Drawing A-401 shows a generic detail but doesn't
specify the membrane termination at the curtain wall mullion. Need to
clarify the flashing sequence and sealant joints.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute professional engineering advice or a binding cost estimate. The project manager and relevant licensed professionals are responsible for verifying all technical details, cost figures, and code compliance before use.
