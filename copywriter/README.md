# Copywriter Plugin for Claude

Capture client voice, generate variations, gate output through approval workflows, stay employed in the AI era.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/copywriter) — AI tools, guides, and weekly digests designed specifically for copywriters.

## Commands

| Command | Description |
|---------|-------------|
| `/brief-intake` | Capture client tone, brand voice, target audience, success metrics, and constraints into a structured brief |
| `/draft-batch` | Generate 3–5 variations of copy with different angles, lengths, and tones for client review |
| `/tone-check` | Analyze copy against the client's brand voice guide and flag deviations with phrase-level evidence |
| `/approval-workflow` | Track client feedback and version history, generate diff summaries, prepare sign-off package |

## Skills

This plugin includes skills that activate automatically when you're working on copywriting tasks:

- **Client Voice Mapping** — Learn and replicate tone from sample copy; produce a Voice Doc reusable across projects
- **Conversion Psychology** — Persuasion hooks, urgency framing, objection handling, and CTA optimization grounded in Cialdini and Sutherland

## Usage examples

```
/brief-intake
New client: SaaS bookkeeping tool for solopreneurs. Tone: warm, plain-spoken,
zero finance jargon. Avoid: "leverage", "synergy", any startup-bro voice.
KPI: free-trial signup. Audience: 1099 freelancers earning $40-120k.
Capture into a brief.
```

```
/draft-batch
Headline + subhead for the homepage hero. Goal: free-trial signup.
Need 5 variants — vary angle (pain vs. gain vs. social proof), length
(short vs. medium), and emotion (calm vs. urgent). Tone per the saved
voice doc.
```

```
/tone-check
Voice doc says: warm, no jargon, sentences under 18 words, "you" not "users".
Copy to check: "Empower your business by leveraging our cutting-edge
automation suite to scale operations." Flag every deviation with evidence.
```

```
/approval-workflow
Round 3 with the client. Their feedback: "love variant B headline, kill
variant C, soften the CTA on D, add a testimonial." Build the diff,
update the version log, prepare the sign-off package.
```

## Disclaimer

Copy generated by this plugin is a draft for copywriter review. Brand voice fidelity, factual claims, and legal/regulatory copy (financial, health, claims-based marketing) must be verified by a qualified copywriter and, where required, a legal or compliance reviewer before publication.
