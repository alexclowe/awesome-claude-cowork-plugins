# Litigation Paralegal Plugin for Claude

Summarize depositions, build case timelines, draft motions, and manage discovery holds.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/paralegal) — AI tools, guides, and weekly digests designed specifically for paralegals.

## Commands

| Command | Description |
|---------|-------------|
| `/deposition-summary` | Extract key admissions, build a topical index, and flag contradictions and impeachable statements from a deposition |
| `/case-timeline` | Build a chronological case timeline from filings, discovery production, and witness statements; flag discovery gaps |
| `/motion-draft` | Template-driven motion writing (motion to dismiss, summary judgment, in limine) with citation slots and jurisdictional rules |
| `/discovery-hold` | Analyze documents for privilege, relevance, and hold scope; generate privilege-log entries |

## Skills

This plugin includes skills that activate automatically when you're working on litigation-related tasks:

- **Legal Research Cite Finder** — Pull cases, statutes, and rules of civil procedure by topic and jurisdiction; flag overruled authorities and circuit splits
- **Evidence Relevance Checker** — Apply Federal Rules of Evidence (relevance, hearsay, character, authentication, chain of custody) to flag admissibility issues

## Usage examples

```
/deposition-summary
Plaintiff's deposition (8 hours, 312 pages). Key topics: notice of defective
condition, prior complaints, repair history. Witness gave conflicting testimony
about when the property manager was first notified. Build topical index and
flag impeachable statements against her June 2024 incident report.
```

```
/case-timeline
Premises liability case, slip-and-fall, NY Supreme. Build timeline from:
2023-08-12 incident, 2024-01-15 complaint filed, 2024-03-04 answer, three
sets of RFPs, 6 deposition dates, plus the incident report and 2022 prior
complaint. Flag any discovery gaps.
```

```
/motion-draft
Motion in limine to exclude defendant's expert opinion on standard of care.
Jurisdiction: SDNY. Grounds: Daubert — expert lacks specialized knowledge
in pediatric nursing and methodology is unreliable. Need cite slots for
Daubert factors and FRE 702.
```

```
/discovery-hold
Review 2,400 emails from custodian (CFO) for privilege and relevance.
Matter: SEC investigation into revenue recognition, FY 2023. Flag attorney-
client communications with outside counsel (Wilson Sonsini), work product,
and any documents responsive to SEC document requests #4 and #7.
```

## Disclaimer

All legal content generated by this plugin is for informational and drafting purposes only. It does not constitute legal advice and does not create an attorney-client relationship. The supervising attorney is responsible for verifying all citations, applying jurisdictional rules, and exercising independent professional judgment before any filing or client communication.
