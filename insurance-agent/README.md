# Insurance Agent Plugin for Claude

Summarize policies, draft renewal letters, and generate claims documentation.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/insurance-agent) — AI tools, guides, and weekly digests designed specifically for insurance agents.

## Commands

| Command | Description |
|---------|-------------|
| `/policy-summary` | Generate client-friendly policy summaries that explain coverage in plain language |
| `/renewal-letter` | Draft personalized renewal recommendation letters with coverage analysis |
| `/claims-doc` | Generate structured claims documentation with coverage analysis and client communication |
| `/client-outreach` | Create personalized client outreach emails for prospecting, retention, and cross-selling |

## Skills

This plugin includes skills that activate automatically when you're working on insurance-related tasks:

- **Insurance Analysis** — Policy analysis across P&C, life, health, and commercial lines with plain-language explanations and regulatory awareness
- **Client Retention** — Client relationship management including annual reviews, cross-selling strategies, retention approaches, and referral building

## Usage examples

```
/policy-summary
HO-3 homeowners policy. Dwelling: $350K, Other structures: $35K, Personal
property: $175K (ACV), Liability: $300K, Med pay: $5K. Deductible: $1,000.
Endorsements: water backup ($10K), scheduled jewelry ($15K ring). Exclusions:
flood, earthquake, mold (standard). Replacement cost on dwelling only.
```

```
/renewal-letter
Client: "James and Lisa Park." Auto policy renewing. Current premium: $2,400/yr
for 2 vehicles. Renewal: $2,760/yr (15% increase). No claims in 3 years.
Increase due to market-wide rate adjustment. Both vehicles have increased
in value. Recommend maintaining current coverage, adding umbrella.
```

```
/claims-doc
Client "Sarah Chen" — tree fell on detached garage during windstorm last
night. Garage partially collapsed, car inside has roof damage. No injuries.
HO-3 policy with $500K dwelling, $50K other structures, $250K personal
property, $1K deductible. Photos taken, tree service contacted.
```

```
/client-outreach
Annual review invitation for "David and Maria Gonzalez." Clients for 5 years,
home + auto bundle. Had a teen turn 16 last year (added to auto). Home value
has increased significantly — haven't reviewed dwelling coverage in 2 years.
No claims history. Want to schedule a 15-minute review call.
```

## Disclaimer

All insurance content generated by this plugin is for informational and drafting purposes only. It does not constitute insurance advice. The agent is responsible for verifying all coverage details, policy terms, and compliance with state-specific regulations before client use.
