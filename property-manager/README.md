# Property Manager Plugin for Claude

Draft tenant notices, board reports, vendor emails, and budget narratives for property management.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/property-manager) — AI tools, guides, and weekly digests designed specifically for property managers.

## Commands

| Command | Description |
|---------|-------------|
| `/tenant-notice` | Draft violation notices, lease renewal letters, move-in/out instructions, and late rent notices with fair housing compliance |
| `/board-report` | Generate board meeting minutes, management reports with financials, occupancy, and maintenance summaries |
| `/vendor-email` | Compose vendor outreach emails, bid requests, service coordination messages, and contract negotiation correspondence |
| `/budget-narrative` | Create budget presentations and financial summaries for board review with variance explanations |

## Skills

This plugin includes skills that activate automatically when you're working on property management tasks:

- **Property Management** — Fair housing law, lease administration, HOA governance, CC&R enforcement, maintenance coordination, and vendor management
- **Board Communication** — Professional board reporting, financial presentation, compliance documentation, and resident communication

## Usage examples

```
/tenant-notice
Draft a lease violation notice for noise complaints. Unit 204, three documented
incidents in 30 days. Community quiet hours are 10pm-7am per CC&Rs Section 5.3.
Tone should be firm but professional. Include cure period and consequences.
```

```
/board-report
Monthly management report for Oakwood HOA, February 2026. Occupancy: 96%.
Delinquencies: $12,400 (down from $18,200). Completed: pool resurfacing ($34K),
lobby painting ($8K). Pending: elevator modernization bid review. Reserve fund
balance: $245,000.
```

```
/vendor-email
Request bids for annual landscape maintenance contract. 120-unit community,
3.5 acres of common area, includes weekly mowing, seasonal planting, irrigation
maintenance, and snow removal. Contract term: 12 months starting April 1.
Need proof of insurance and three references.
```

```
/budget-narrative
2026 operating budget narrative for Maple Ridge Condominiums. Total budget:
$1.2M (up 6% from 2025). Key variances: insurance +18% ($42K increase),
utilities +8%, landscaping flat. Reserve contribution increased to 15% per
reserve study recommendation. No special assessment proposed.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute legal advice. The property manager is responsible for verifying all content against applicable fair housing laws, local ordinances, and governing documents before use.
