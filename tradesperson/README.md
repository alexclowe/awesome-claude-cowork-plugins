# Tradesperson Plugin for Claude

Draft customer estimates, write inspection reports, and generate scopes of work.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/tradesperson) — AI tools, guides, and weekly digests designed specifically for tradespeople.

## Commands

| Command | Description |
|---------|-------------|
| `/estimate` | Generate customer estimates with line items for labor, materials, permits, and warranties across plumbing, electrical, HVAC, and general trades |
| `/inspection-report` | Create structured inspection reports from jobsite notes with findings, recommendations, photo references, and code citations |
| `/scope-of-work` | Draft detailed SOW proposals with phases, materials, timeline, and payment schedule |
| `/customer-email` | Write follow-ups, warranty documentation, maintenance reminders, and estimate explanations |

## Skills

This plugin includes skills that activate automatically when you're working on trade-related tasks:

- **Trade Documentation** — Building code awareness, permit references, industry terminology for plumbing, electrical, HVAC, and safety documentation
- **Customer Relations** — Professional trade communication, estimate presentation, warranty explanation, and upsell framing

## Usage examples

```
/estimate
Estimate for a full bathroom remodel. Scope: remove existing tub and tile, install
walk-in shower with glass enclosure, new vanity and toilet, retile floor. House
built in 1985, may need updated plumbing. Customer in Austin, TX. Include permit
line item and 1-year workmanship warranty.
```

```
/inspection-report
Inspection report for residential electrical panel. Found: double-tapped breakers
on slots 4 and 7, Federal Pacific Stab-Lok panel (known fire hazard), no AFCI
protection on bedroom circuits, aluminum wiring on 3 branch circuits. House is
1972 build, 100A service. Recommend full panel replacement and AFCI retrofit.
```

```
/scope-of-work
SOW for commercial HVAC replacement. Building is 4,000 sq ft office space, replacing
two 5-ton rooftop units with high-efficiency Carrier units. Include ductwork
inspection/repair, new programmable thermostats, refrigerant line replacement.
Project needs to be completed over two weekends to minimize business disruption.
Payment schedule: 40% deposit, 40% at rough-in, 20% at completion.
```

```
/customer-email
Follow-up email to a customer who received a plumbing estimate last week for a
whole-house repipe (copper to PEX). They seemed concerned about the cost ($12,500).
Help me explain the value, mention the 25-year warranty on PEX, and offer a
financing option through GreenSky.
```

## Disclaimer

All content generated by this plugin is for drafting and planning purposes only. It does not constitute engineering advice or official code compliance verification. The tradesperson is responsible for verifying all building code references, permit requirements, material specifications, and pricing accuracy before use with customers or for official documentation.
