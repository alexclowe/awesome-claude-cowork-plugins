# Sales Rep Plugin for Claude

Research prospects, draft follow-up sequences, and generate sales proposals.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/sales-rep) — AI tools, guides, and weekly digests designed specifically for sales professionals.

## Commands

| Command | Description |
|---------|-------------|
| `/prospect-brief` | Generate research summaries with company intel, decision-maker profiles, talking points, and recent news |
| `/follow-up` | Draft email sequences for post-meeting, post-demo, nurture, and re-engagement scenarios |
| `/sales-proposal` | Create proposals with value proposition, pricing, ROI analysis, and competitive positioning |
| `/call-prep` | Build pre-call research briefs with agenda, discovery questions, and objection handling |

## Skills

This plugin includes skills that activate automatically when you're working on sales-related tasks:

- **Sales Strategy** — Consultative selling, MEDDIC/BANT frameworks, value-based positioning, objection handling, and pipeline management
- **Sales Communication** — Professional persuasive writing, personalization, urgency without pressure, and CTA best practices

## Usage examples

```
/prospect-brief
Research brief for Acme Corp. They're a mid-market SaaS company, ~500 employees,
recently raised Series C. I'm targeting the VP of Engineering. We sell developer
productivity tools. Find talking points around their tech stack and growth challenges.
```

```
/follow-up
Post-demo follow-up sequence for a prospect who saw our analytics platform demo
yesterday. They expressed interest in the real-time dashboard feature but had
concerns about integration with their existing Salesforce setup. Decision timeline
is 30 days.
```

```
/sales-proposal
Proposal for a 200-seat enterprise license of our project management platform.
Prospect is currently using a competitor (Monday.com). Key pain points: lack of
resource allocation features, poor reporting. Annual contract value target: $48K.
Include ROI analysis based on 15% productivity improvement.
```

```
/call-prep
Pre-call brief for discovery call with CFO of a healthcare company (800 employees).
They downloaded our whitepaper on cost optimization. We sell expense management
software. Help me prepare an agenda, discovery questions, and likely objections
around implementation timeline and security compliance.
```

## Disclaimer

All content generated by this plugin is for drafting and planning purposes only. Sales professionals are responsible for verifying all company information, pricing accuracy, and compliance with their organization's sales policies before use in client communications.
