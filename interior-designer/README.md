# Interior Designer Plugin for Claude

Write project proposals, draft spec sheets, and create client presentations.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/interior-designer) — AI tools, guides, and weekly digests designed specifically for interior designers.

## Commands

| Command | Description |
|---------|-------------|
| `/project-proposal` | Generate design proposals with scope of work, concept overview, phasing, fee structure, and terms |
| `/spec-sheet` | Draft FF&E schedules and material specifications with manufacturer details, dimensions, and lead times |
| `/concept-narrative` | Create design concept descriptions for client presentations with color palette rationale and mood descriptions |
| `/client-update` | Draft project status updates, change order explanations, budget revisions, and milestone reports |

## Skills

This plugin includes skills that activate automatically when you're working on interior design tasks:

- **Interior Design** — Space planning, color theory, material knowledge, FF&E specification standards, building codes awareness, and design documentation
- **Design Client Management** — Proposal writing, scope management, change order communication, budget discussions, and managing client expectations

## Usage examples

```
/project-proposal
Residential renovation for a 2,400 sq ft mid-century modern home in Austin, TX.
Client wants to update kitchen, primary bedroom, and two bathrooms. Budget is
$180K–$220K. They want to preserve the original architectural character while
modernizing functionality.
```

```
/spec-sheet
Create an FF&E schedule for a boutique hotel lobby — 1,800 sq ft. Design style
is contemporary coastal. Need specifications for: reception desk, lounge seating
(2 sofas, 4 accent chairs), coffee tables, area rugs, pendant lighting, and
artwork. Budget tier is mid-to-high.
```

```
/concept-narrative
Write a concept narrative for a 3,200 sq ft law firm office redesign. Design
direction: modern professional with warm accents. Inspired by mid-century
Scandinavian design. Key materials: white oak, brass, bouclé, and dark green
marble. The goal is to feel approachable yet authoritative.
```

```
/client-update
Project: Williams Residence Kitchen Renovation. Phase 2 (demolition and rough-in)
is complete, 3 days ahead of schedule. Custom cabinetry lead time has extended by
2 weeks due to supplier delay. Need to communicate timeline impact and suggest
options — either substitute a comparable cabinet line or adjust the schedule.
```

## Disclaimer

All proposals, specifications, and design documentation generated by this plugin are drafts for professional review only. They do not constitute final construction documents, licensed design services, or contractual agreements. The interior designer is responsible for verifying all specifications, codes, pricing, and project details before client submission or implementation.
