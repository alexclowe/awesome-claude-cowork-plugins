# Mortgage Broker Plugin for Claude

Generate rate comparisons, draft pre-approval letters, and create client communications for mortgage lending.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/mortgage-broker) — AI tools, guides, and weekly digests designed specifically for mortgage brokers.

## Commands

| Command | Description |
|---------|-------------|
| `/rate-comparison` | Generate side-by-side rate comparisons across loan products with APR, monthly payment, and total cost breakdowns |
| `/pre-approval-letter` | Draft pre-approval letters with borrower qualifications, loan terms, and compliance language |
| `/client-update` | Create status update communications for borrowers at each stage of the loan process |
| `/refinance-analysis` | Produce refinance scenario analyses comparing current vs. proposed terms with break-even timelines |

## Skills

This plugin includes skills that activate automatically when you're working on mortgage-related tasks:

- **Mortgage Lending** — Loan products, rate structures, underwriting guidelines, compliance requirements, and closing procedures
- **Client Communication** — Borrower education, rate explanations, closing process walkthroughs, and milestone updates

## Usage examples

```
/rate-comparison
Compare 30-year fixed at 6.75%, 15-year fixed at 6.125%, and 7/1 ARM at 6.25%
for a $425,000 purchase with 20% down. Borrower wants to see monthly payment
differences and total interest paid over the life of each loan.
```

```
/pre-approval-letter
Pre-approval letter for John and Sarah Martinez. Approved up to $550,000.
Combined income $185,000/yr. Credit scores 760/745. 20% down payment from
savings. Conventional loan, 30-year fixed. Valid for 90 days.
```

```
/client-update
Loan status update for the Nguyen family. We submitted to underwriting on
Monday. Appraisal came back at $475,000 — matches purchase price. Still
waiting on employment verification from borrower's HR department. Expected
closing date March 28.
```

```
/refinance-analysis
Refinance analysis for client. Current loan: 30-year fixed at 7.25%, $380,000
balance, 26 years remaining. Proposed: 30-year fixed at 6.375%, rolling in
$4,200 closing costs. Client wants to know monthly savings, break-even point,
and total cost comparison.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute financial advice, mortgage advice, or a commitment to lend. The mortgage professional is responsible for verifying all figures, ensuring regulatory compliance, and exercising independent professional judgment before client use.
