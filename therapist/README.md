# Therapist Plugin for Claude

Draft session notes, build treatment plans, write pre-authorization letters, generate client correspondence, and run HIPAA, outcome-monitoring, and treatment-plan-versioning workflows.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/therapist) — AI tools, guides, and weekly digests designed specifically for therapists.

## Commands

| Command | Description |
|---------|-------------|
| `/session-note` | Draft SOAP, DAP, or BIRP session notes with format selection guidance |
| `/treatment-plan` | Build structured treatment plans with measurable goals, objectives, interventions, and timelines |
| `/pre-auth` | Generate insurance pre-authorization and medical necessity letters with CPT/ICD-10 references |
| `/client-letter` | Draft progress letters, discharge summaries, and referral letters |
| `/hipaa-check` | Flag potential HIPAA privacy violations in session notes before sharing or AI-tool input |
| `/measure-therapy-outcomes` | Aggregate PHQ-9, GAD-7, PCL-5 (and other) scores over sessions and format for licensing-board audits |
| `/treatment-plan-versioning` | Track treatment-plan changes over time with diffs, justifications, and clinical reasoning |

## Skills

This plugin includes skills that activate automatically when you're working on therapy-related tasks:

- **Clinical Mental Health** — DSM-5 diagnostic criteria, evidence-based therapy modalities (CBT, DBT, EMDR, ACT, psychodynamic), treatment planning, and progress measurement
- **Therapeutic Communication** — Empathetic clinical writing, health literacy adaptation, and crisis sensitivity
- **Suicide Risk Assessment Protocol** — Columbia C-SSRS and SAFE-T frameworks; auto-prompts a structured risk assessment when warning indicators appear in clinical content
- **Trauma-Informed Language Guardrails** — Flags re-traumatizing or pathologizing phrasing and suggests SAMHSA-aligned alternatives

## Usage examples

```
/session-note
DAP note for individual session with adult client. Presenting issue: generalized anxiety
with panic attacks 2-3x weekly. Session focused on CBT psychoeducation about the
anxiety cycle and introduced diaphragmatic breathing. Client practiced in session and
reported reduction in acute distress from 8/10 to 4/10.
```

```
/treatment-plan
Treatment plan for 32yo female diagnosed with Major Depressive Disorder, recurrent,
moderate (F33.1). Client reports persistent low mood, anhedonia, sleep disruption,
and difficulty concentrating at work. Previous trial of sertraline with partial
response. Client is motivated for therapy. Plan to use CBT with behavioral activation.
```

```
/hipaa-check
Scan this excerpt before I bring it to consultation:
"Client J.S., age 34, software engineer at Stripe in SF, came in 4/15/26 after
her brother — ER doc at UCSF — convinced her to seek treatment. She mentioned
the specific incident at her old role at Square in 2023..."
Destination: peer consultation group (no BAA in place with the platform we use).
```

```
/measure-therapy-outcomes
Aggregate the following 12 sessions for J.S. — PCL-5 scores: baseline 62, then
58, 55, 51, 49, 47, 44, 41, 38, 35, 33, 31. PHQ-9: baseline 18, then 16, 15,
14, 13, 12, 11, 10, 9, 8, 7, 7. C-SSRS ideation level 2 weeks 1-3, level 1
weeks 4-12, no behaviors. Format for chart and for an upcoming board audit.
```

```
/treatment-plan-versioning
Update the prior treatment plan (dated 1/15/26). Changes: PCL-5 has dropped
from 62 to 48 (RCI achieved). Adding EMDR Phase 2-3 work targeting the index
trauma; reducing CPT homework load; increasing session frequency from biweekly
to weekly during EMDR processing phase. Client agreed in session today.
```

```
/client-letter
Progress letter for client's disability attorney. Client has been in treatment for
12 months for Bipolar I Disorder (F31.1) with comorbid GAD. Include functional
limitations, treatment compliance, and current symptom presentation.
```

## Disclaimer

All clinical content generated by this plugin is for drafting purposes only and requires review by a licensed mental health professional. It does not constitute clinical advice, diagnosis, or treatment. The clinician is responsible for verifying all clinical information, exercising independent professional judgment, and ensuring compliance with HIPAA and applicable state regulations before use. **AI tools without a Business Associate Agreement (BAA) cannot receive PHI — verify your tool's BAA status before pasting clinical content.**
