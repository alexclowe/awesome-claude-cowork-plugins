# Speech-Language Pathologist Plugin for Claude

Draft therapy notes, write IEP goals, and generate progress reports and therapy materials.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/speech-language-pathologist) — AI tools, guides, and weekly digests designed specifically for speech-language pathologists.

## Commands

| Command | Description |
|---------|-------------|
| `/therapy-note` | Draft SOAP session notes covering articulation, language, fluency, voice, and AAC |
| `/iep-goal` | Generate SMART goals aligned to state standards with measurable criteria and baseline data |
| `/progress-report` | Create progress monitoring reports with data analysis, graph descriptions, and recommendations |
| `/therapy-material` | Design custom therapy activities, word lists, sentence strips, and conversation scripts |

## Skills

This plugin includes skills that activate automatically when you're working on speech-language pathology tasks:

- **Speech-Language Pathology** — Articulation, phonology, language disorders, fluency, voice, AAC, dysphagia, and cognitive-communication expertise
- **Educational Documentation** — IEP writing, IDEA compliance, SMART goal frameworks, progress monitoring methods, and parent-friendly reporting

## Usage examples

```
/therapy-note
SOAP note for 6yo male, articulation therapy session. Working on /r/ in initial
position of words. Produced /r/ correctly in 7/10 trials at word level with visual
cue (mirror). Continues to substitute /w/ for /r/ in conversational speech.
Used minimal pairs activity (rake/wake, red/wed). Good attention and motivation
today.
```

```
/iep-goal
IEP goals for a 9yo female with expressive language disorder. Current baseline:
MLU of 3.2 morphemes (age expectation 7.5+), difficulty with complex sentences,
limited use of conjunctions and subordinate clauses. She is in 3rd grade, struggles
with written narrative and oral retelling. Needs 3 measurable annual goals.
```

```
/progress-report
Progress report for a 5yo male receiving articulation therapy for /s/, /z/, and
/th/ sounds. Treatment started September 2025. Current data: /s/ initial — 85%
accuracy at sentence level (up from 40%), /s/ final — 60% at word level, /z/ —
45% at word level, /th/ — not yet targeted. Attends twice weekly, 30-minute
sessions. Include recommendations for continued therapy.
```

```
/therapy-material
Create a therapy activity for a group of 4 students (ages 7-8) working on
following multi-step directions. Target: 3-step directions with temporal and
spatial concepts (before/after, next to/between). Make it a game-based activity
that can run for 20 minutes. Include 15 direction cards at increasing complexity.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute clinical or educational advice. The speech-language pathologist is responsible for verifying all clinical information, ensuring compliance with applicable laws and regulations, and exercising independent professional judgment before clinical or educational use.
