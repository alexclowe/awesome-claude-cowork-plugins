# The AI Career Lab — Claude Cowork Plugins

Profession-specific AI plugins for [Claude Cowork](https://claude.com/cowork). Each plugin adds slash commands and domain skills tailored to how you actually work — not generic AI, but tools built for your specific profession.

**41 slash commands. 20 domain skills. 10 professions.**

## Quick Start

In Claude Cowork, run:

```
/plugin marketplace add alexclowe/theaicareerlab-plugins
```

Then install any plugin:

```
/plugin install pharmacist@aicareerlab-plugins
```

## What's Inside Each Plugin

Every plugin includes:

- **Slash commands** — Run `/command-name` to generate profession-specific documents, notes, and communications. Each command includes structured prompts with the right context for your field.
- **Domain skills** — Background expertise that Claude applies automatically when working on tasks in your profession. Skills cover both technical knowledge and communication patterns.
