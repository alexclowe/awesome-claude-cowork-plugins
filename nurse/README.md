# Nurse Plugin for Claude

Draft nursing notes, create shift handoffs, build care plans, and generate patient education materials.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/nurse) — AI tools, guides, and weekly digests designed specifically for nurses.

## Commands

| Command | Description |
|---------|-------------|
| `/nursing-note` | Draft clinical documentation with structured assessment, intervention, and evaluation sections |
| `/shift-handoff` | Generate SBAR format handoff reports for safe transitions of care |
| `/care-plan` | Build nursing care plans with NANDA-I diagnoses, NIC interventions, and NOC outcomes |
| `/patient-education` | Create plain-language patient education materials for diverse health literacy levels |

## Skills

This plugin includes skills that activate automatically when you're working on nursing-related tasks:

- **Clinical Nursing** — Nursing assessment, NANDA-I taxonomy, NIC/NOC classification, medication administration documentation, and vital signs interpretation
- **Patient Education** — Health literacy-adapted materials, teach-back method guidance, and discharge instructions

## Usage examples

```
/nursing-note
Nursing note for 68yo male, post-op day 1 s/p right total knee arthroplasty. VS:
T 37.2, HR 78, BP 138/82, RR 16, SpO2 97% RA. Pain 6/10 right knee, managed with
PCA morphine. Surgical dressing CDI. Foley catheter draining clear yellow urine.
PT eval completed, ambulated 20 ft with walker. Tolerating clear liquids.
```

```
/shift-handoff
SBAR handoff for 54yo female admitted for CHF exacerbation. Background: EF 30%,
h/o A-fib, CKD stage 3. Currently on IV furosemide 40mg BID, metoprolol 25mg BID,
lisinopril 10mg daily. I&O last 12h: intake 800mL, output 1400mL. BMP this AM
showed K+ 3.3 — MD notified, KCl 40mEq PO ordered. Weight down 1.2kg from admission.
```

```
/care-plan
Care plan for 45yo female newly diagnosed with Type 2 Diabetes, A1c 9.2%.
Started on metformin 500mg BID. BMI 34. Reports minimal understanding of diabetes
management. Lives alone, works night shifts. Needs comprehensive education on
blood glucose monitoring, medication management, diet, and sick day rules.
```

```
/patient-education
Patient education handout for new diagnosis of atrial fibrillation. Patient is
72yo male starting on apixaban (Eliquid) 5mg BID and metoprolol succinate 50mg
daily. Needs to understand what A-fib is, why blood thinners are needed, signs
to watch for, and when to go to the ER.
```

## Disclaimer

All clinical content generated by this plugin is for drafting purposes only and requires review by a licensed nursing professional. It does not constitute medical advice or replace clinical judgment. The nurse is responsible for verifying all clinical information, following facility-specific protocols, and ensuring compliance with HIPAA and applicable state nursing practice acts before use.
