# Personal Trainer Plugin for Claude

Build workout programs, write nutrition plans, and generate client reports.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/personal-trainer) — AI tools, guides, and weekly digests designed specifically for personal trainers.

## Commands

| Command | Description |
|---------|-------------|
| `/workout-program` | Generate periodized training programs with exercise selection, sets/reps/rest, and progression plans |
| `/nutrition-plan` | Create macro-based nutrition guides with meal ideas, timing, and hydration guidelines |
| `/progress-report` | Draft client progress summaries with body composition, performance metrics, and program adjustments |
| `/fitness-content` | Generate social media posts, email newsletters, exercise tutorials, and educational content for fitness brands |

## Skills

This plugin includes skills that activate automatically when you're working on fitness-related tasks:

- **Exercise Science** — Program design principles, periodization models, biomechanics, injury prevention, and evidence-based training methodology
- **Fitness Business** — Client retention strategies, pricing models, social media marketing, referral programs, and business growth for solo fitness professionals

## Usage examples

```
/workout-program
Design a 4-day upper/lower split for an intermediate lifter. Goal is hypertrophy.
Client is 32yo male, 180lbs, no injuries. Include warm-up, cool-down, and a
4-week progression plan.
```

```
/nutrition-plan
Create a fat loss nutrition plan for a 28yo female, 155lbs, 5'6", moderately
active (trains 4x/week). She is lactose intolerant and prefers Mediterranean-style
meals. Target: 1lb/week loss.
```

```
/progress-report
8-week progress summary for male client, 40yo. Starting: 210lbs, 28% BF,
bench 135x5. Current: 203lbs, 24% BF, bench 165x5. Adherence: trained 29 of
32 scheduled sessions. Goal is continued fat loss while building strength.
```

```
/fitness-content
Write a week of Instagram content (5 posts) for a personal trainer who
specializes in strength training for women over 40. Tone: encouraging,
evidence-based, no bro-science. Include captions, hashtags, and CTA for
each post.
```

## Disclaimer

All fitness and nutrition content generated by this plugin is for educational and drafting purposes only. It does not constitute medical advice. Clients should consult a qualified healthcare provider before starting any exercise or nutrition program, especially if they have existing medical conditions, injuries, or are pregnant.
