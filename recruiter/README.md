# Recruiter Plugin for Claude

Write job descriptions, draft candidate outreach, and build interview scorecards.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/recruiter) — AI tools, guides, and weekly digests designed specifically for recruiters.

## Commands

| Command | Description |
|---------|-------------|
| `/job-description` | Generate inclusive, well-structured job descriptions with DEI-conscious language and clear requirements |
| `/candidate-outreach` | Draft personalized sourcing messages for email, LinkedIn InMail, and follow-up sequences |
| `/interview-scorecard` | Create structured interview scorecards with role-specific competencies and behavioral questions |
| `/offer-letter` | Draft offer letters with compensation breakdowns and generate offer comparison matrices |

## Skills

This plugin includes skills that activate automatically when you're working on recruiting-related tasks:

- **Talent Acquisition** — Sourcing strategy, candidate evaluation, structured hiring, employment law awareness, and compensation benchmarking
- **Employer Branding** — EVP communication, DEI-inclusive language, candidate experience, role marketing, and brand voice consistency

## Usage examples

```
/job-description
Write a job description for a Senior Backend Engineer at a Series B fintech startup.
Tech stack: Go, PostgreSQL, Kubernetes. Remote-first, US time zones.
Salary range: $170K–$210K + equity. Must-haves: 5+ years backend experience,
distributed systems. Nice-to-haves: fintech domain knowledge, open source contributions.
```

```
/candidate-outreach
Draft a LinkedIn InMail for a passive candidate — she's a Staff Engineer at Stripe
with 8 years of experience in payments infrastructure. We're hiring a VP of Engineering
at a Series A payments startup. Mention her recent conference talk on event-driven
architecture. Include 2 follow-up messages spaced 5 and 10 days apart.
```

```
/interview-scorecard
Create an interview scorecard for a Product Manager role (mid-level, B2B SaaS).
Key competencies: product sense, analytical thinking, cross-functional collaboration,
customer empathy. Include 2 behavioral questions per competency in STAR format,
a 1–5 rating scale, and a debrief summary template.
```

```
/offer-letter
Draft an offer letter for a Senior Data Scientist. Base: $185K, signing bonus: $20K,
equity: 0.15% over 4 years with 1-year cliff. Benefits: health/dental/vision,
401(k) with 4% match, unlimited PTO, $2K annual learning stipend.
Start date: April 15. Also generate a comparison matrix — the candidate is evaluating
offers from us, a FAANG company, and a late-stage startup.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute legal advice. Employment law varies by jurisdiction — the recruiter is responsible for ensuring all job descriptions, outreach, and offer letters comply with applicable federal, state, and local employment laws (including EEOC, ADA, FLSA, and local pay transparency requirements).
