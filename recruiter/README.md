# Recruiter Plugin for Claude

Write job descriptions, draft candidate outreach, and screen for ghost jobs and AI-written resumes.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/recruiter) — AI tools, guides, and weekly digests designed specifically for recruiters.

## Commands

| Command | Description |
|---------|-------------|
| `/job-description` | Generate inclusive, well-structured job descriptions with DEI-conscious language and clear requirements |
| `/candidate-outreach` | Draft personalized sourcing messages for email, LinkedIn InMail, and follow-up sequences |
| `/interview-scorecard` | Create structured interview scorecards with role-specific competencies and behavioral questions |
| `/offer-letter` | Draft offer letters with compensation breakdowns and generate offer comparison matrices |
| `/ghost-job-detect` | Analyze a job posting for ghost-job red flags and return a probability score with evidence |
| `/ai-resume-flag` | Detect AI-written resume language patterns and flag for human review with specific evidence |
| `/compensation-band-check` | Cross-check a job description's stated comp band against role, seniority, and market data |

## Skills

This plugin includes skills that activate automatically when you're working on recruiting-related tasks:

- **Talent Acquisition** — Sourcing strategy, candidate evaluation, structured hiring, employment law awareness, and compensation benchmarking
- **Employer Branding** — EVP communication, DEI-inclusive language, candidate experience, role marketing, and brand voice consistency
- **Ghost Job Detection** — Auto-activates ghost-job red-flag detection on job postings, ATS pipeline indicators, and pay-transparency gaps
- **AI-Resume Detector** — Pattern recognition for LLM-generated resume text — sentence length variance, em-dash density, and generic accomplishment phrasing

## Usage examples

```
/ghost-job-detect
Analyze this posting for ghost-job risk: "Senior Software Engineer — Remote US.
We are seeking a passionate, driven engineer to join our growing team. 10+ years
required across modern web stacks. Salary: competitive. Apply through our talent
community to be considered for current and future opportunities." Posting age:
67 days. No req ID, no hiring manager named.
```

```
/ai-resume-flag
Screen this resume for AI-authorship signals. Candidate is applying for a Senior
Backend Engineer role. Bullets include: "Spearheaded scalable, performant, and
reliable microservices — driving transformative outcomes across the organization,"
"Leveraged cutting-edge cloud infrastructure to deliver 50% efficiency gains,"
"Orchestrated cross-functional initiatives ensuring strategic alignment." No
specific tools, named projects, or stakeholders mentioned.
```

```
/compensation-band-check
Pressure-test this band before posting: Staff Software Engineer, infra/platform,
Series C SaaS (~$120M ARR), San Francisco hybrid 2 days/week. Proposed: base $210K,
target bonus 10%, equity 0.08% over 4 years (no refresh defined), standard benefits.
We're targeting candidates currently at Stripe, Datadog, Snowflake.
```

```
/job-description
Write a job description for a Senior Backend Engineer at a Series B fintech startup.
Tech stack: Go, PostgreSQL, Kubernetes. Remote-first, US time zones.
Salary range: $170K–$210K + equity. Must-haves: 5+ years backend experience,
distributed systems. Nice-to-haves: fintech domain knowledge, open source contributions.
```

```
/candidate-outreach
Draft a LinkedIn InMail for a passive candidate — she's a Staff Engineer at Stripe
with 8 years of experience in payments infrastructure. We're hiring a VP of Engineering
at a Series A payments startup. Mention her recent conference talk on event-driven
architecture. Include 2 follow-up messages spaced 5 and 10 days apart.
```

```
/interview-scorecard
Create an interview scorecard for a Product Manager role (mid-level, B2B SaaS).
Key competencies: product sense, analytical thinking, cross-functional collaboration,
customer empathy. Include 2 behavioral questions per competency in STAR format,
a 1–5 rating scale, and a debrief summary template.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute legal advice. Employment law varies by jurisdiction — the recruiter is responsible for ensuring all job descriptions, outreach, screening practices, and offer letters comply with applicable federal, state, and local employment laws (including EEOC, ADA, FLSA, NYC Local Law 144, and local pay transparency requirements).
