# Optometrist Plugin for Claude

Document eye exams, draft referral letters, create patient education materials, and write insurance narratives.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/optometrist) — AI tools, guides, and weekly digests designed specifically for optometrists.

## Commands

| Command | Description |
|---------|-------------|
| `/exam-note` | Document comprehensive eye exams with visual acuity, refraction, slit lamp, fundoscopy, tonometry, diagnosis, and plan |
| `/referral-letter` | Draft specialist referral letters to ophthalmologists with clinical findings, imaging results, and specific questions |
| `/patient-education` | Create vision care education materials — condition explanations, lens care, post-dilation instructions, and screen time guidance |
| `/insurance-narrative` | Write medical necessity letters for contact lens fittings, specialty lenses, vision therapy, and procedures |

## Skills

This plugin includes skills that activate automatically when you're working on optometry-related tasks:

- **Clinical Optometry** — Comprehensive eye examination, ocular disease assessment, contact lens fitting, refractive assessment, ICD-10 coding for eye conditions, and CPT coding for eye procedures
- **Patient Communication** — Vision care education, treatment option explanation, compliance counseling, and pediatric/geriatric communication adaptation

## Usage examples

```
/exam-note
Comprehensive eye exam. Patient: 54yo female, chief complaint blurry near
vision x6 months. VA: OD 20/25, OS 20/30, OU 20/25. Refraction: OD -1.50
-0.75x180, OS -2.00 -0.50x175, ADD +2.00. SLE: trace nuclear sclerosis OU,
otherwise unremarkable. IOP: OD 16, OS 17. Dilated fundus: C/D 0.3 OU,
macula flat, periphery intact. Assessment: presbyopia, mild nuclear cataract.
```

```
/referral-letter
Referral to retina specialist. Patient: 62yo male with type 2 diabetes x15
years, HbA1c 8.2%. Dilated exam shows moderate non-proliferative diabetic
retinopathy OU with clinically significant macular edema OS. OCT shows
central subfield thickness 340 microns OS. Requesting evaluation for
anti-VEGF therapy.
```

```
/patient-education
Patient education for new daily disposable contact lens wearer. 16yo female,
first-time CL fit. Rx: -3.25 OD, -2.75 OS. Need instructions for insertion/
removal, hygiene, wearing schedule buildup, and when to remove lenses and
call the office. Parent will be present for teaching.
```

```
/insurance-narrative
Medical necessity letter for scleral contact lens fitting. Patient: 38yo
with keratoconus OU, status post corneal crosslinking OS. BCVA with
spectacles: OD 20/60, OS 20/50. BCVA with scleral lenses: OD 20/25,
OS 20/20. Patient cannot achieve functional vision with spectacles.
Previous soft toric CL trial unsuccessful — unstable fit with irregular cornea.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute medical or optometric advice. The optometrist is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
