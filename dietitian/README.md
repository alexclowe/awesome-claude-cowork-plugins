# Dietitian Plugin for Claude

Generate nutrition assessments, meal plans, progress notes, and client education materials for dietitians.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/dietitian) — AI tools, guides, and weekly digests designed specifically for dietitians.

## Commands

| Command | Description |
|---------|-------------|
| `/nutrition-assessment` | Draft comprehensive nutrition assessments with dietary analysis, anthropometric data, and clinical findings |
| `/meal-plan` | Generate individualized meal plans with macronutrient targets, recipes, and dietary modification guidance |
| `/progress-note` | Create progress notes documenting dietary adherence, lab trends, weight changes, and plan adjustments |
| `/client-education` | Produce patient education handouts on specific dietary topics, conditions, or lifestyle modifications |

## Skills

This plugin includes skills that activate automatically when you're working on nutrition-related tasks:

- **Clinical Nutrition** — Medical nutrition therapy, macronutrient calculations, lab interpretation, dietary guidelines, and condition-specific protocols
- **Dietary Communication** — Client education, meal planning communication, behavior change counseling, and motivational strategies

## Usage examples

```
/nutrition-assessment
Nutrition assessment for 52yo male, newly diagnosed T2DM. A1C 8.2%,
BMI 31.4, BP 142/88. Currently eating fast food 4-5x/week, drinks 3-4
sodas daily, skips breakfast. Reports no food allergies. Takes metformin
500mg BID. Goal: improve glycemic control and lose 15 lbs over 6 months.
Sedentary job, willing to start walking.
```

```
/meal-plan
7-day meal plan for 34yo female athlete training for a marathon. Needs
2,800 kcal/day, high carb (55%), moderate protein (20%), moderate fat (25%).
Lactose intolerant, prefers Mediterranean-style eating. Needs pre-run and
post-run fueling strategies. Include hydration guidance for long training
runs.
```

```
/progress-note
Progress note for 45yo female with CKD stage 3. Seen 4 weeks after initial
consult. Lab update: GFR improved from 42 to 46, phosphorus down from 5.1
to 4.6, potassium stable at 4.8. Reports following low-sodium diet well
but struggling with protein portion control. Weight down 3 lbs. Review
current plan and adjust.
```

```
/client-education
Client education handout on the DASH diet for a 60yo male with hypertension.
Keep language at 6th grade reading level. Include a sample daily menu, a
list of high-sodium foods to avoid, and practical tips for reducing sodium
when eating out. Patient speaks English as a second language — use simple
sentences and avoid idioms.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute medical nutrition therapy or dietary advice. The registered dietitian is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
