# ESG Sustainability Analyst Plugin for Claude

Extract ESG data, score risk, draft disclosures, and benchmark sustainability performance.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/esg-sustainability-analyst) — AI tools, guides, and weekly digests designed specifically for ESG and sustainability analysts.

## Commands

| Command | Description |
|---------|-------------|
| `/extract-esg-data` | Ingest a corporate report and extract ESG KPIs (carbon, diversity, governance) mapped to GRI/SASB/TCFD/CSRD |
| `/assess-esg-risk` | Map a company against ESG risk frameworks, flag material issues per SASB materiality map, and generate a risk score |
| `/draft-esg-disclosure` | Generate SEC Rule 10b5-1 disclosures, ESG proxy filings, and sustainability report drafts in CSRD format |
| `/track-esg-performance` | Ingest historical data, benchmark against a peer set, and track progress toward stated targets |

## Skills

This plugin includes skills that activate automatically when you're working on ESG and sustainability tasks:

- **Regulatory Standard Mapping** — Auto-map ESG initiatives to GRI/SASB/TCFD/CSRD/EU Taxonomy/SFDR requirements
- **Carbon Accounting Automation** — Ingest Scope 1/2/3 inputs, calculate footprint per the GHG Protocol, and flag data-quality issues

## Usage examples

```
/extract-esg-data
Pasted text from a 2024 annual sustainability report (mid-cap industrial,
NAICS 333). Extract Scope 1/2/3 emissions, gender diversity at board and
exec level, board independence %, and any TCFD-aligned scenario disclosures.
Map each KPI to the matching GRI, SASB, and TCFD reference.
```

```
/assess-esg-risk
Target: a US-listed regional bank, $48B AUM, ~6,500 employees. We're
underwriting a green bond. Run a SASB-materiality screen, flag the top
five material ESG risks for commercial banking, and score governance
on a 1-5 scale with rationale.
```

```
/draft-esg-disclosure
Need a CSRD-format double-materiality narrative for our parent EU subsidiary
covering climate transition risk, water stress at three manufacturing sites,
and supply-chain human-rights due diligence. Include the ESRS cross-references
and a placeholder data table for each topical standard.
```

```
/track-esg-performance
Historical Scope 1+2 emissions: 142,000 / 138,500 / 131,200 / 125,400 tCO2e
(FY21–FY24). SBTi-validated 2030 target is -42% vs 2020 baseline (155,000
tCO2e). Benchmark against three named peers and tell me whether we're on
trajectory.
```

## Disclaimer

All ESG and sustainability content generated by this plugin is for informational and drafting purposes only. It does not constitute legal, accounting, audit, or investment advice. ESG analysts and reporting professionals are responsible for verifying all data, framework citations, and disclosure language against the source documents and the latest version of each standard (GRI, SASB, TCFD, CSRD/ESRS, EU Taxonomy, SFDR, SEC) before filing or publication. Outputs may not reflect rule amendments after the model's knowledge cutoff.
