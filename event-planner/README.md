# Event Planner Plugin for Claude

Create event proposals, vendor outreach, day-of timelines, and client updates for event planning.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/event-planner) — AI tools, guides, and weekly digests designed specifically for event planners.

## Commands

| Command | Description |
|---------|-------------|
| `/event-proposal` | Draft client proposals with event concept, scope, budget breakdown, timeline, and terms |
| `/vendor-outreach` | Compose vendor inquiry emails, RFP templates, contract negotiation, and coordination emails |
| `/event-timeline` | Build detailed day-of timelines with vendor assignments, setup/teardown, contact info, and contingencies |
| `/client-update` | Generate progress reports, budget updates, change summaries, and milestone completions |

## Skills

This plugin includes skills that activate automatically when you're working on event planning tasks:

- **Event Design** — Event concept development, logistics planning, vendor coordination, budget management, and timeline creation
- **Client Management** — Proposal writing, expectation management, change communication, and post-event reporting

## Usage examples

```
/event-proposal
Corporate annual gala for 250 guests, downtown venue, black-tie optional.
Budget range: $75K-$100K. Client wants live band, plated dinner, awards
ceremony segment, and photo booth. Date: September 20, 2026. Need full
proposal with concept, scope, budget breakdown, and payment terms.
```

```
/vendor-outreach
RFP for wedding floral design. Event: 180-guest wedding, October 10, 2026.
Venue: outdoor garden ceremony + indoor ballroom reception. Need: bridal
bouquet, 6 bridesmaid bouquets, 20 centerpieces, ceremony arch florals,
cocktail hour arrangements. Color palette: dusty rose, sage, ivory.
Budget allocation for florals: $8,000-$12,000.
```

```
/event-timeline
Day-of timeline for a 200-guest wedding. Ceremony at 4:00 PM, cocktail
hour 4:30-5:30, reception 5:30-10:00. Vendors: caterer, DJ, florist,
photographer, videographer, officiant, hair/makeup. Include setup starting
at 10 AM, vendor load-in times, and teardown by midnight.
```

```
/client-update
Monthly progress report for the Johnson-Lee wedding, October 10, 2026.
Completed: venue contract signed, caterer booked, save-the-dates mailed.
In progress: floral vendor selection (3 proposals received), DJ booking.
Budget spent to date: $32,000 of $85,000. Next milestones: finalize
floral vendor by March 30, send invitations by June 1.
```

## Disclaimer

All content generated by this plugin is for informational and drafting purposes only. It does not constitute legal or contractual advice. The event planner is responsible for verifying all details, confirming vendor availability, and reviewing contractual terms before client or vendor submission.
